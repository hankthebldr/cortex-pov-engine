One-line install

Stage-1 cradles are tiny, hand-auditable, and verify the rest of the install against the release manifest's SHA-256 digests.

Linux Ubuntu 22.04 / Debian 12 / RHEL 9 
curl -fsSL https://github.com/__GH_OWNER__/__GH_REPO__/releases/latest/download/install.sh | sudo bash

Verify before you run

Every release ships SHA256SUMS and a structured manifest.json. Audit, then execute.

curl -fsSLO https://github.com/__GH_OWNER__/__GH_REPO__/releases/latest/download/install.sh
curl -fsSLO https://github.com/__GH_OWNER__/__GH_REPO__/releases/latest/download/SHA256SUMS
sha256sum --check --ignore-missing SHA256SUMS
# install.sh: OK
sudo bash install.sh
