# scenarios/edr

EDR plane scenarios — endpoint detection via Cortex XDR agent.

These scenarios exercise Cortex XDR agent detection capabilities including process BIOC rules,
memory injection analytics, persistence mechanism detection, and IOC matching against known-bad
hashes, IPs, and domains.
