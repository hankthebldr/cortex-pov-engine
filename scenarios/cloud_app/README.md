# scenarios/cloud_app

Cloud App Security scenarios — Shadow IT, OAuth abuse, CI/CD attacks.

These scenarios exercise Cortex Cloud App Security detection capabilities targeting
SaaS application abuse, OAuth token theft, CI/CD pipeline attacks, and ASPM (Application
Security Posture Management) signal generation.

Primary source repo: gocortexbrokenbank (intentionally vulnerable CI/CD app)

Use case prefix: UCS-CAPP-xx
